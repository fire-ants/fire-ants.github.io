So what is Hitter Value (HVal)?

Hitter Value (HVal) is an analytics algorithm developed by the Fire Ants.  This algorithm uses a combination of Qualitiative and Quantitative scores to assign a value per pitch.

Quantititave Scores are based on the typical outcomes of a pitch, using 1.0 as for on-base and -1.0 for an out. For example, Strikes are -.0333, Balls are 0.025 and so forth. 

Qualitative Scores are based on text processing from the offical scorer's statement about the pitch and At Bat event. For example, balls put into play with key words such as "sharply" or "liner" are scored more heavily towards the batter's benefit.  Conversely, when the scorer describes balls in play as "softly", "pops", or "flies", then these are scored more in favor of the pitcher.

Hitter Value is a sum of these Quantitative and Qualitative scores.  HVal = QualScore + QuantScore.
